# Modern DevOps Weather App

A simple microservices-based weather application built with **Docker** and **Docker Compose**.  

The project includes:

- **Auth Service**: Handles user authentication  
- **Weather Service**: Fetches weather data from an API  
- **UI Service**: Frontend for interacting with the services  
- **MySQL Database**: Stores user authentication data
